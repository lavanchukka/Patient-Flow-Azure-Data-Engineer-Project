In this project we'll create real time healthcare patient flow data pipeline as data source using python and demonstrate the use of various services like Azure Eventhubs, Azure Databricks, Delta lake and synapse analytics. Also, implement medallion architecture, schema evolution, create facts and dimension tables and connect the cleaned and transformed data to PowerBI.

We will perform the following tasks:
• Real-time data streaming with Azure Event Hubs + Kafka
• Ingestion & transformations with Databricks + PySpark
• Implement the Medallion Architecture (Bronze, Silver, Gold)
• Schema evolution in Delta Lake
• SCD Type 2 handling in Gold layer
• Building a Star Schema (Fact & Dimension tables)
• Loading data into Azure Synapse SQL Pool
• Connecting Power BI for visualizations
